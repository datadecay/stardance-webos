Guide

To open apps click the app's icon on the desktop

Open the "Dev" app and upload your zipfile. The app will install and you can use it :)

In Conf app, open personalization settings

To package apps, create a zip file containing the structure (names are dependent on config.json):

app.zip/
├── app.png
├── config.json
├── app.html
└── app.js

config.json contains the app's configuration. Example:

{
  "id": "read",
  "name": "Read :3",
  "version": "1.0.0",
  "description": "read stuff?",
  "icon": "read.png",
  "html": "read.html",
  "script": "read.js",
  "width": "600px",
  "height": "450px"
}

app.png, app.html, and app.js are kinda self explanatory

Once you have your zip file, open the Dev app and upload it. If your config is correct, it should install and work :3
